# Wireframe: MILAI Playground (5 Screens)

Black/white wireframes for coding education platform. Use only black, white, grays. Simple shapes, lines, placeholders.

## Screen 1: Problem Statement
**Sidebar (250px):** Logo "M" circle, nav: Home, Learn Web Dev, Practice, Placement, Reports. Bottom: "Student" icon.

**Content:** Header "New Learning Challenge" + lightbulb + subtitle. "Problem Statement" text: "Build responsive Task Manager with CRUD, task filtering." Tech tags: ReactJS, NodeJS, ExpressJS, MongoDB (4 rectangles). "Skills" section: 2 checkmark bullets (Frontend, Backend). Button: "Start Building in Playground".

## Screen 2: Create Project
**Sidebar:** Same as Screen 1.

**Content:** Header "Create Your Project". Form: Project Name input ("Task Manager"), Tech Stack (4 tags), Problem Statement (read-only textarea), Description (textarea). Toggle "Enable AI Assistance" (on) + 4 bullets. Button: "Create Project & Start Coding".

**Right Panel (300px):** "AI Mentor" title. List: Real-time review, Generate snippets, Learning feedback, Progress tracking, Project planning. Dark gray bg.

## Screen 3: Coding Workspace
**Left Sidebar (50px):** Icon bar (file, search, settings).

**Top Bar:** Tabs: server.js (active), index.js, App.js. Right: Save, Run, Share, profile icon.

**Editor (top-left):** Line numbers + code: `import express from 'express'; const app = express(); app.use(express.json()); app.listen(PORT, () => { console.log('Server running'); });` Dark bg.

**Terminal (bottom-left):** Black window: "Starting server...", "Connected to MongoDB...", "Server running on port 5000."

**Right Panel - AI Chat (400px):** Header "AI Mentor" + avatar. Chat: User (right, dark): "Stuck with database connection?" AI (left, light): "Check MongoDB connection..." User: "Using Mongoose..." AI: "Missing useNewUrlParser..." Quick Actions (2x2): Fix Bug, Refactor, Explain, Test. Progress circle "85%". Tip box. Input: "Ask AI Mentor..." + mic.

## Screen 4: Project Results
**Sidebar:** Same as Screen 1.

**Content:** Header: Checkmark + "Project Submitted Successfully!" + subtitle. Tabs: Evaluation Results (active), Learning Feedback, Progress Report. Status: Circle progress "70%". "What's Working" (light gray): 4 checks (structured, CRUD, responsive, errors). "What's Missing" (medium gray): 3 warnings (API errors, validation, auth). Collapsible: Architecture, Code Quality, Tests, Performance, Security. Skills: React.js 4.5★, JavaScript 4★.

## Screen 5: Learning Journey
**Sidebar:** Same as Screen 1.

**Content:** Header "Continue Learning Journey" + subtitle. 5 skill cards: API Handling, Frontend, Testing, Data Management, Performance. Each: icon, name, description, "Learn More" button. Learning Path: 3-stage progress bar (Completed, Skills, Next Project) - first 2 dark, last light. Stage circles. Button: "Go to My Learning Path".

## Specs
**Colors:** Black #000, White #FFF, Grays #F5F5F5/#CCC/#666/#333.

**Typography:** Sans-serif. Headers bold/large, body regular/medium, labels small/gray, code monospace.

**Elements:** Rectangles (buttons/inputs), Circles (icons/progress), Lines (dividers), Dashed placeholders.

**Spacing:** Padding 16/24/32px, Margins 24/32px.

**Interactive:** Buttons (dark/bold), Inputs (rect+placeholder), Toggles (circle on track), Tabs (active underlined), Progress (partial).

**Notes:** Show 5 screens horizontal/separate. Consistent sidebar. Wireframe conventions. Desktop view. Clear sections. Scrollable=dashed borders.
